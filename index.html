First Steps

Master the basics of hardware control with simple but satisfying projects.

  • Blink an LED (the "Hello World" of hardware)
  • LDR sensor experiments (light sensitivity)
  • Try different LED patterns and timings
  • Understanding digital vs analog signals
C++ (Arduino) #define LED_PIN 2 #define LDR_PIN A0 void setup() { pinMode(LED_PIN, OUTPUT); Serial.begin(115200); } void loop() { int lightLevel = analogRead(LDR_PIN); if (lightLevel < 500) { digitalWrite(LED_PIN, HIGH); // Turn on in darkness } else { digitalWrite(LED_PIN, LOW); // Turn off in light } delay(100); }
