🔌 Day 3: Hardware with ESP32

Bring your code to life with physical hardware! Learn to control LEDs, read sensors, and create interactive IoT projects.

💡

First Steps

Master the basics of hardware control with simple but satisfying projects.

  • Blink an LED (the "Hello World" of hardware)
  • LDR sensor experiments (light sensitivity)
  • Try different LED patterns and timings
  • Understanding digital vs analog signals
C++ (Arduino) #define LED_PIN 2 #define LDR_PIN A0 void setup() { pinMode(LED_PIN, OUTPUT); Serial.begin(115200); } void loop() { int lightLevel = analogRead(LDR_PIN); if (lightLevel < 500) { digitalWrite(LED_PIN, HIGH); // Turn on in darkness } else { digitalWrite(LED_PIN, LOW); // Turn off in light } delay(100); }

Grand Finale

Create an interactive IoT music player that responds to remote commands!

  • Connect a buzzer and play music
  • Use ChatGPT to generate new songs
  • Connect to a server
  • Let people pick which audio to play remotely
  • Build a complete IoT music system
C++ (Arduino) #include <WiFi.h> #define BUZZER_PIN 25 // Musical notes (frequencies in Hz) const int C4 = 262, D4 = 294, E4 = 330, F4 = 349; void playTwinkleTwinkle() { int melody[] = {C4, C4, G4, G4, A4, A4, G4}; int durations[] = {4, 4, 4, 4, 4, 4, 2}; for (int i = 0; i < 7; i++) { tone(BUZZER_PIN, melody[i], 1000 / durations[i]); delay(1200 / durations[i]); } }
